ThreadSense for Facebook — User Guide

Version 0.1.0

ThreadSense is a Chrome extension that analyzes posts and comments you can already see on Facebook desktop and shows local heuristic risk estimates for troll-like behavior and fake-account patterns. It runs entirely in your browser and does not send data to any server.

Important: ThreadSense never claims anyone is a troll, bot, or fake account. Scores are pattern-based guesses and can be wrong. Use them as one input among many—not as proof.

1. What ThreadSense does

While you browse Facebook (feed, posts, groups, etc.), ThreadSense:

  • Finds visible posts and comments in the page.
  • Extracts comment/post text and optional public profile cues.
  • Scores each item with two independent heuristics:
    • Troll-behavior risk
